# Synthesis Patterns

Patterns for synthesizing multi-source research findings.

## Pattern 1: Consensus Synthesis

Use when sources largely agree.

```markdown
## [Topic]: Key Findings

### Consensus View
[What most sources agree on] [1, 2, 3]

### Supporting Evidence
- [Evidence point 1] [1, 2]
- [Evidence point 2] [2, 3]
- [Evidence point 3] [1, 3]

### Minor Variations
[Small differences in how sources frame the issue]

### Confidence: HIGH
Sources in strong agreement.
```

## Pattern 2: Divergence Synthesis

Use when sources disagree significantly.

```markdown
## [Topic]: Competing Views

### View A: [Position Name]
**Supported by**: [Sources 1, 2]
**Key arguments**:
- [Argument 1]
- [Argument 2]
**Evidence strength**: [Strong/Moderate/Weak]

### View B: [Position Name]
**Supported by**: [Sources 3, 4]
**Key arguments**:
- [Argument 1]
- [Argument 2]
**Evidence strength**: [Strong/Moderate/Weak]

### Assessment
[Which view has stronger evidence and why]

[Note any factors that might explain the disagreement:
- Different time periods studied
- Different methodologies
- Different definitions
- Potential biases]

### Confidence: MEDIUM
Topic contested among authoritative sources.
```

## Pattern 3: Gap-Aware Synthesis

Use when information is incomplete.

```markdown
## [Topic]: Current Understanding

### What We Know (High Confidence)
[Well-established findings] [Sources]

### What We Think (Medium Confidence)
[Probable but not fully verified] [Limited sources]

### What Remains Unknown
- [Gap 1] - No sources address this
- [Gap 2] - Conflicting/insufficient evidence
- [Gap 3] - Topic too recent for research

### Implications
[What the gaps mean for the research question]

### Suggested Next Steps
- [How to fill Gap 1]
- [How to fill Gap 2]
```

## Pattern 4: Hierarchical Synthesis

Use for complex topics with multiple layers.

```markdown
## [Topic]: Layered Analysis

### High-Level Summary
[1 paragraph executive summary]

### Primary Findings
1. **[Finding 1]**: [Summary] [1, 2, 3]
2. **[Finding 2]**: [Summary] [4, 5]
3. **[Finding 3]**: [Summary] [6, 7, 8]

### Detailed Analysis

#### [Finding 1] Deep Dive
[Expanded analysis with full citations]

#### [Finding 2] Deep Dive
[Expanded analysis with full citations]

### Cross-Cutting Themes
[Patterns that appear across multiple findings]

### Confidence by Layer
- High-level summary: [Confidence]
- Individual findings: [Varies]
- Detailed analysis: [Varies]
```

## Pattern 5: Temporal Synthesis

Use when tracking evolution over time.

```markdown
## [Topic]: Evolution of Understanding

### Historical Context
[How this topic was understood previously] [older sources]

### Current State
[Present understanding] [recent sources]

### Emerging Trends
[What's changing] [most recent sources]

### Timeline
- [Year]: [Key development]
- [Year]: [Key development]
- [Year]: [Current state]

### Trajectory Assessment
[Where this seems to be heading based on trends]
```

## Citation Integration Rules

### Inline Citations
Use for specific claims:
```markdown
Studies show 40% improvement in productivity [1, 2].
```

### Attribution Phrases
Use for opinions or interpretations:
```markdown
According to McKinsey [3], the market will...
Researchers at MIT argue that... [4]
```

### Block Citations
Use for extended quotes or key passages:
```markdown
As noted in the 2024 Gartner report:

> "Direct quote from the source that captures
> an important point." [5]
```

## Multi-Source Integration

When combining 3+ sources on same point:

1. **Identify common ground** - What do all sources agree on?
2. **Note variations** - How do they differ in framing/detail?
3. **Assess weight** - Which sources are most authoritative?
4. **Synthesize** - Create unified statement with citations
5. **Note caveats** - Any important nuances?

Example:
```markdown
Research consistently shows [general finding] [1, 2, 3],
though estimates vary from [low] [1] to [high] [3].
The most recent analysis suggests [specific finding] [2].
```

## Quality Markers

### Strong Synthesis Indicators
- Clear thesis/main point
- All claims cited
- Disagreements acknowledged
- Confidence levels stated
- Gaps identified

### Weak Synthesis Indicators
- Vague generalizations
- Missing citations
- Cherry-picked sources
- Overconfident conclusions
- Ignored contradictions
